# 部署指南

## 环境要求
1. JDK 21
2. MySQL 
3. Intellij IDEA
4. Navicat Premium
5. Git

## 部署步骤
1. 克隆代码
2. 创建MySQL数据库
3. 修改配置文件
4. 运行项目
5. 测试接口

## 常见问题
### 问题1：端口8080被占用
**症状**: `Web server failed to start. Port 8080 was already in use.`
**解决方案**:
     修改 application.properties

     server.port=8081
### 问题2：数据库连接失败
**症状**: `Access denied for user 'root'@'localhost`

**解决方案**:
1. 检查MySQL服务是否启动

2. 验证用户名密码是否正确

3. 确认数据库是否存在
### 问题1：CORS跨域错误
**症状**: ` 前端调用API时出现跨域错误`

**解决方案**:

   确认WebConfig.java中已配置前端地址
